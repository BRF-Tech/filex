filex

Self-hosted file management for the example.com stack.

Features

Multi-driver storage
local, S3, FTP, SFTP, WebDAV.
Live thumbnails
ffmpeg + ghostscript + libreoffice.
Replica
Path-glob fan-out + reconcile.
Persistent queue
sqlite, redis, postgres.

Quick install

docker run -p 5212:5212 ghcr.io/brf-tech/filex:latest

Embed snippet

import { mountFileExplorer } from '@brftech/filex-core';
mountFileExplorer('#root', { api: { baseURL: '/api/files' } });
